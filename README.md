# 📈 Predictive Sales AI

An AI-powered sales analysis and forecasting application. Accepts Excel spreadsheets and generates deep profitability insights using Gemini 2.5 Flash.

## ✨ Features
- **Predictive Scoring**: 0-100 score based on revenue trends, profitability, and more.
- **Profitability Analysis**: COGS efficiency, margin trend detection, and product-level margin deep dives.
- **AI Forecasting**: Next-period projections for revenue, profit, and margins.
- **Risk & Opportunity Mapping**: Automated detection of customer concentration and regional trends.
- **PDF Export**: Download professional reports instantly.

## 🛠️ Tech Stack
- **Backend**: Node.js, Express, ExcelJS, Gemini AI
- **Frontend**: Vanilla JS, HTML5, CSS3 (Glassmorphism)
- **Validation**: Zod

## 🚀 Quick Start
1. Clone the repository.
2. Install dependencies: \`npm install\`
3. Add your \`GEMINI_API_KEY\` to a \`.env\` file.
4. Run the server: \`npm start\`
5. Open \`http://localhost:3000\`
